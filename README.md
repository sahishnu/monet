# Monet Digital Studio

Static landing page for a web development agency: **Astro** (SSG), **React** (sections + mobile nav), **Tailwind CSS**.

## Commands

```bash
npm install
npm run dev
npm run build   # output: dist/
npm run preview
```

## Push to a new remote

Git cannot be initialized from some automated environments. On your machine:

```bash
chmod +x scripts/git-init-and-push.sh
./scripts/git-init-and-push.sh
```

Then add `origin` and push, or use `gh repo create ... --push` as printed by the script.
